# org.kde.desktop.PanelZoom
panel for charging plasmoid wavetask

How to install:
- Download the code
- Unzip and rename the folder to org.kde.desktop.PanelZoom
- Create a folder called layout-templates in /home/yourusername/.local/share/plasma/
- Copy the org.kde.desktop.PanelZoom folder to /home/yourusername/.local/share/plasma/layout-templates/
- That's it.
